## Display Modes

- Velocity  
- Airspeed  
- Fuel in Stage  
- Electrical Charge in Vessel  
- Altitude  
- Time Until Next Apoapsis and Periapsis  
- Delta-V Remaining  
- Burn Time Remaining  
- Current Sphere of Influence  
- Vessel Name + Crew Count
